### 1) 项目名称

**AstraTickets｜企业级 AI 智能客服工单系统**
副标题：从“来单→分类→检索→AI首答→人工协同→统计看板”全链路落地

------

### 2) 主要技术栈 / 工具 / 平台

- **后端**：Python（FastAPI / Uvicorn / Pydantic / SQLAlchemy），RESTful + OpenAPI/Swagger
- **前端**：React + Vite + Ant Design（或 Tailwind），会话式聊天 UI + 工单管理台
- **数据库**：PostgreSQL/MySQL（事务 + 索引优化），Redis（会话/限流/缓存）
- **向量检索 / RAG**：Chroma/FAISS（可替换 Weaviate/PGVector），LangChain / LlamaIndex
- **大模型**：OpenAI / DeepSeek / Qwen / Llama（可云端，也可本地推理）
- **消息队列/异步**：Celery/Redis（异步分类、批量嵌入、离线任务）
- **鉴权与安全**：JWT/OAuth2、RBAC、审计日志、敏感信息脱敏
- **可观测性**：结构化日志、Prometheus + Grafana（或 CloudWatch）、Sentry
- **测试与质量**：Pytest、Playwright（前端）、黑盒/白盒用例、LLM 提示词回归测试
- **工程化/部署**：Docker Compose、GitHub Actions（CI/CD）、Railway/Render/AWS（任选其一）
- **周边工具**：Postman/Thunder Client、Makefile、一键启动脚本、数据脱敏脚本

------

### 4) 一句话介绍

随着大模型的普及，企业在客服环节越来越关注 **智能化、降本增效**。本项目将带领学员从零开始，设计并开发一个 **基于Python和RAG检索增强的智能客服工单系统**。您将学习如何搭建完整的 **后端服务（FastAPI）+ 前端管理台（React/AntD）+ 数据库（MySQL/Redis）**，并结合 **大语言模型（OpenAI/国产大模型）**，实现工单自动分类、知识库智能问答、AI生成初步回复以及运营数据可视化。

完成本项目后，您不仅能掌握 **企业级RAG应用的落地方法**，还将产出一个可放入简历的完整项目案例，提升在互联网/AI/数据平台方向面试中的竞争力。
