# 课程大纲

| 节次                                   | 教学主题                                                     | 核心内容与学习成果                                           |
| -------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **第 1 节：项目启动与架构设计**        | - 项目总体介绍与演示- 技术选型（FastAPI、SQLite、React、Chroma）- 环境搭建与项目结构初始化 | 完成本地开发环境配置，理解系统架构（前后端 + 数据库 + AI 模块） |
| **第 2 节：后端基础与工单模型设计**    | - FastAPI 路由与依赖注入- 设计工单数据库模型（用户、工单、回复表）- 实现工单增删改查 API | 实现基础 RESTful 接口，掌握 ORM（SQLAlchemy）操作            |
| **第 3 节：前端界面与工单管理模块**    | - 使用 React + Ant Design 构建前端界面- 工单列表、创建、详情页开发- 前后端联调 | 完成可视化工单管理台（包含创建、查看、过滤、分页）           |
| **第 4 节：引入本地 RAG 检索增强模块** | - 介绍 RAG 基本概念- 使用 **SentenceTransformers + ChromaDB** 构建知识库- 文档切片、向量化、相似度检索 | 学会搭建离线文档知识库与向量召回模块（全程本地运行）         |
| **第 5 节：AI 智能分类与回复生成**     | - 训练轻量文本分类模型（例如 scikit-learn + TF-IDF）- 使用 **本地大模型/开源推理接口**（如 Ollama + Llama2 / Qwen）生成初步回复- 分类+回复融合逻辑 | 工单自动分配到类别，并生成“AI 初步答复”示例                  |
| **第 6 节：对话上下文与人工协同机制**  | - 工单与上下文对话存储- 结合前端展示“AI 建议 + 人工修改”- 聊天式交互界面 | 打造混合式客服界面（AI + 人类协同）                          |
| **第 7 节：统计看板与系统优化**        | - FastAPI 后端聚合查询- 使用 ECharts 绘制工单趋势、响应时间、满意度- 添加用户登录鉴权（JWT） | 构建可视化数据看板与用户系统，强化实用性                     |
| **第 8 节：部署上线与项目总结**        | - Docker Compose 打包部署- 使用 Streamlit 或 Railway 免费托管- 项目总结 + 简历写法指导 | 一键部署可运行 Demo，导出中英文简历项目描述模板              |
